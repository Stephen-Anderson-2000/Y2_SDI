As the project is further developed, the following features will be added\+:


\begin{DoxyItemize}
\item Ability to export annotated images, annotation data, and label lists to a file.
\item Ability to import annotation data and label lists from files.
\item Ability to input custom label names.
\item Ability to create \& edit annotations by clicking \& dragging on the image.
\item Ability to edit existing labels and annotations.
\item Ability to search through \& sort images, labels, and annotations.
